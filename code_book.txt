A codebook for variables in "tidy_data.txt" data files.
A codebook for "tidy_data_agg.txt" data file is simillar but with the remaining variables averaged to activity-subject pairs.

Activity: type of activity performed during measurment (factor 1-6)
	1 WALKING

	2 WALKING_UPSTAIRS
	
3 WALKING_DOWNSTAIRS
	
4 SITTING

	5 STANDING
	
6 LAYING


Subject: identifyier of a volounteer performing an activity (factor 1-30)

Remaining variables: summary (mean, std) normalized measurements of all variables (numeric -1 - 1).
	Total number of these variables is 66.
	More specific information can be obtained from the "features_info.txt" file.
